= The Typed Imp Language Specification
:toc: left
:toclevels: 2
:sectnums:
:icons: font

Typed Imp is a small typed imperative language. Its syntax covers
integer and boolean literals, variables, arithmetic and boolean
operators, sequencing, conditionals, `while` loops, and typed variable
declarations. The static semantics assigns types to expressions and
commands under a typing context; the dynamic semantics evaluates
expressions and executes commands under an environment that maps
variables to values.

== Syntax

The abstract syntax of Typed Imp distinguishes types, literals,
variables, expressions, commands, and whole programs.

${syntax: type literal id expr command prog}
